# Tamil Nadu - Higher Secondary Education - Second Year - Computer Science Programs

This repository contains the Python and SQL programs I practiced from the **Tamil Nadu State Board Higher Secondary Education Second Year (Class 12) Computer Science textbook for 2024–2025**.

I wrote, debugged, and tested nearly all the programs covered in the textbook. The files are organized by their original units and chapters, along with the CSV, database, chart, and C++ integration files used by the programs.

## Achievement

I scored a **centum (full marks) in Computer Science** in my Class 12 public examination, following my centum in Class 11. This marked two consecutive centums in Computer Science. I uploaded this collection to GitHub as a personal archive and a memory of my school programming journey.
